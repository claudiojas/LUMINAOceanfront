# CONTEXT.md - Projeto Lumina Oceanfront (Portfólio Real Estate)

**Objetivo:** Criar uma Landing Page de Alto Padrão para venda de imóveis de luxo, demonstrando superioridade visual e técnica sobre PDFs estáticos.

## 1. Stack Tecnológica
* **Framework:** React (Vite)
* **Estilização:** Tailwind CSS
* **Animações:** Framer Motion (Essencial para o ar de "luxo").
* **Ícones:** Lucide React (Traço fino/light).

## 2. Design System (Luxo Minimalista)
* **Cores:**
    * Base: `#FAFAFA` (Quase branco).
    * Texto Principal: `#1C1917` (Stone 900 - Preto suave).
    * Accent (Detalhes): `#C2410C` (Laranja Queimado/Cobre) ou `#DAA520` (Dourado fosco).
* **Tipografia:**
    * Headings (Títulos): 'Playfair Display' (Serifa, elegante).
    * Body (Texto): 'Lato' ou 'Montserrat' (Sans-serif, limpo).

## 3. Estrutura da Página (Seções)

1.  **Hero Imersivo:**
    * Imagem de fundo full-screen (Fachada do prédio).
    * Título elegante: "Onde o horizonte encontra o seu lar."
    * Badge sutil: "Lançamento Exclusivo".

2.  **O Conceito (Storytelling):**
    * Texto curto sobre estilo de vida, alinhado com uma foto de interior.
    * Efeito de *Fade In* ao rolar.

3.  **Galeria (Carrossel):**
    * Swipe (arrastar) suave no celular.
    * Fotos: Piscina, Living, Suíte Master.

4.  **Diferenciais (Grid):**
    * Ícones finos e elegantes.
    * Ex: "Concierge 24h", "Piscina Infinita", "Automação Residencial".

5.  **Plantas Inteligentes (O Diferencial Técnico):**
    * Abas: [Apartamento Garden] | [Penthouse].
    * Ao clicar, troca a imagem da planta instantaneamente com uma transição suave.

6.  **Localização & CTA:**
    * Mapa estilizado (imagem estática tons de cinza).
    * Formulário de captura elegante (Nome, WhatsApp).

7.  **Sticky Bar (Mobile Only):**
    * Barra fixa no rodapé com botão "Falar com Corretor" (WhatsApp).
